# MSDK developer sample app

To start working with sample app, add the following SDK files to the root folder:
* OPPWAMobile.xcframework
* OPPWAMobile_MSA.xcframework
* ipworks3ds_sdk.xcframework

# Configuration
Basic constants can be changed in Config.swift file.
Sample app covers only basic scenarios. You are welcome to adapt sample code to your specific needs.

# Server integration
Please note, that the app uses our test integration server (OPPWAMobile_MSA.xcframework). Please adapt it to use your own backend API in your app.
